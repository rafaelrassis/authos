FAQ

O que é o Authos?
Authos é site disponível tanto para computador quanto pra smartphone. Aqui sua criança vai se desenvolver de uma maneira descontraída com a ajuda de seu psicólogo.

O que devo fazer se encontrar problemas técnicos na plataforma?
Se você encontrar problemas técnicos, entre em contato com nosso suporte técnico pelo e-mail suporte@authos.com.

Como faço para me cadastrar?
Você pode se cadastrar clicando no botão “Criar conta” na página inicial e preenchendo os dados solicitados. Você precisará fornecer informações básicas e criar uma senha para sua conta.

O que é o código de identificação e como ele funciona?
O código de identificação é um número único atribuído a cada paciente da plataforma Authos. Esse código serve para garantir a segurança e a privacidade das informações pessoais e para facilitar que o profissional encontre a conta de seu paciente.
